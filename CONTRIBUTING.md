### Prerequisites

Before you begin, please ensure you have the following installed:
* [Terraform](https://www.terraform.io/downloads.html) >= 1.0
* [Yandex Cloud Provider](https://registry.terraform.io/providers/yandex-cloud/yandex/latest) >= 0.72.0

### Contribution Workflow

1.  **Fork the repository** to your own GitHub account.
2.  **Create a new branch** for your changes (e.g., `git checkout -b feature/my-new-feature` or `fix/my-bug-fix`).
3.  **Make your changes.** Please ensure your code is formatted using `terraform fmt`.
4.  **Update Examples:** If you are changing the module's source code, please be sure to make any appropriate updates to the relevant files in the [examples](https://github.com/terraform-yacloud-modules/terraform-yandex-cdn/tree/main/examples) folder.
5.  **Commit your changes** with a clear and descriptive commit message.
6.  **Push your branch** to your fork on GitHub.
7.  **Open a Pull Request (PR)** from your fork's branch to the `main` branch of this repository.
8.  **Link the issue** you are addressing in your PR description (e.g., `Closes #82`).
9.  Wait for a review from the maintainers.
